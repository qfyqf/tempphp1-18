实验简介：由于企业生存与发展内外环境的不断变化，企业难以避免危机事件的发生，如果不能有效的开展危机管理应对，将直接影响企业的生存与发展；危机沟通作为企业危机管理应对的重要一环，良好的危机沟通能有效降低企业危机的冲击，使企业转危为安。培养企业危机沟通的应对能力已成为工商管理、市场营销等管理专业学生必备技能。

申报材料
企业危机具有不可复制性和处置紧迫性等特点，传统实验难以再现真实的 企业危机事件现场，针对企业危机沟通能力的培养基本停留在使用文字、图片、 案例讲述的教学方式，学生无法切实体会企业面临危机事件的紧迫感和效果反 馈的及时性，导致学生主动学习的积极性不高，理论知识掌握不深刻，实际动 手能力差。为充分提高学生的学习兴趣和效果，有效掌握企业危机管理沟通的 相关理论和运用能力，本项目综合应用多媒体、三维建模、人机交互、大数据、 人工智能等数字化、智能化技术，虚拟仿真企业危机发生的情境，模拟训练学 生开展企业危机管理沟通组织与应对的全过程，帮助学生掌握企业危机事件评 估方法、危机沟通的组织、公告发布、召开新闻发布会与应对记者提问、危机 面谈的技巧与方法以及危机沟通的事后总结与反思工作，使学生确实具备成功 开展企业危机管理沟通工作的组织与实践能力。
